# Nestle_Financial_Model
# Nestle India Ltd (NESTLEIND) - Equity Analysis & Custom Financial Model

## Project Overview

This project showcases an automated approach to financial data extraction and analysis for **Nestle India Ltd**, leveraging a proprietary Excel-based financial model integrated with the Screener.in platform.

The primary objective was to streamline the equity research process by defining a custom set of data points (including key ratios, valuation metrics, and standardized financial statement layouts) and automating their extraction, significantly reducing manual data entry and formatting time.

## Files Included

*   **`Screener_Custom_Template.xlsx`**: The original Excel template file that defines the proprietary data structure and calculated metrics. This file is used to configure the Screener.in download feature.
*   **`NESTLEIND_Financial_Data.xlsx`**: A sample of the actual data downloaded for Nestle India using the custom template. This file serves as the input for further in-depth valuation models (e.g., DCF, Comparable Analysis).

## Methodology

1.  **Custom Template Design**: A comprehensive Excel template was designed to include all necessary data points and derived metrics for standardized analysis.
2.  **Platform Integration**: The template was uploaded to the user account on Screener.in, linking the download function to the custom format.
3.  **Automated Extraction**: The financial data for NESTLEIND was exported to the custom format, ensuring consistency and accuracy.
4.  **Analysis**: The extracted data is ready for immediate analysis, enabling faster decision-making and efficient financial modeling.

## Key Skills Demonstrated

*   **Data Automation**: Expertise in using technology platforms to streamline data extraction processes.

*   **Time Efficiency**: Reduced data collection and preparation time for a single stock analysis by **~80%**.
*   **Standardization**: Ensured consistent data presentation across all analyzed companies.
